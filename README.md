🔄 Inversão de Nomes em C#
📌 Descrição

Este programa em C# solicita ao usuário quatro nomes e, em seguida, inverte a ordem desses nomes, exibindo o resultado no console.

⚙️ Funcionalidades
Recebe 4 nomes digitados pelo usuário
Realiza a inversão da ordem dos nomes manualmente
Exibe os nomes na ordem invertida
Aguarda uma tecla para encerrar o programa
🔁 Lógica de Funcionamento

O programa utiliza uma variável auxiliar (auxiliar) para realizar a troca dos valores:

O primeiro nome troca de posição com o quarto
O segundo nome troca de posição com o terceiro
💻 Como executar
Instale o .NET SDK
Salve o código em um arquivo Program.cs
Abra o terminal na pasta do projeto
Execute:
dotnet run
🖥️ Exemplo de uso
Digite o nome#1: Ana
Digite o nome#2: João
Digite o nome#3: Maria
Digite o nome#4: Pedro

Nomes inseridos na sequência Invertida:
Pedro
Maria
João
Ana
📝 Observações
O programa utiliza Console.ReadLine() para entrada de dados
A inversão é feita manualmente, sem uso de arrays ou listas
Console.ReadKey() mantém o console aberto até o usuário pressionar uma tecla
🚀 Possíveis melhorias
Utilizar arrays para facilitar a manipulação dos nomes
Permitir inserir mais de 4 nomes
Validar entradas vazias
Criar uma versão com ordenação automática
